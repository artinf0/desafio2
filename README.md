
# Como executar o projeto

## Back end
Pré-requisitos: Java 8

```bash
# clonar repositório
git clone https://github.com/gabrielribeirojb/projeto-sds3.git

# entrar na pasta do projeto back end
cd backend

# executar o projeto
./mvnw spring-boot:run
```
